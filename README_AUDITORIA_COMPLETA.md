## 🆘 Si Hay Problemas

### Build falla
1. Check: Node version matches requirement
2. Clear: `rm node_modules && npm install`
3. Rebuild: `npm run build`

### Tests fallan
1. Check: Lint warnings (formatter, no es bloqueante)
2. Ignore: Prettier format warnings
3. Fix: TypeScript errors si hay (no debería haber)

### Visual issues
1. Check: Tailwind CSS compiled
2. Check: Browser cache cleared
3. Compare: Contra screenshots en TESTING_CHECKLIST.md

### Accesibility issues
1. Check: ARIA labels en lugar correcto
2. Use: Axe DevTools para validar
3. Compare: Contra WCAG 2.1 AA requirements
